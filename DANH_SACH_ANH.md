## ✅ HƯỚNG DẪN SỬ DỤNG

1. **Tải ảnh về** và đặt tên đúng như danh sách trên
2. **Copy vào thư mục** tương ứng:
   - Ảnh biển → `public/images/destinations/`
   - Ảnh làng nghề → `public/images/crafts/`
   - Ảnh ẩm thực → `public/images/food/`
3. **Chạy lại web**: `npm run dev`
4. **Kiểm tra** tại http://localhost:3000

**Lưu ý:** 
- Tên file PHẢI CHÍNH XÁC (không dấu, viết thường, đúng số)
- Định dạng: `.jpg` hoặc `.png`
- Kích thước đề xuất: 800x600px
- Dung lượng: dưới 500KB mỗi ảnh
